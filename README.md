
# Page Replacement Visualizer (Minimal)

A single-page micro-project to visualize Page Replacement Algorithms: FIFO, LRU, Optimal.

## Features
- Input reference string and number of frames.
- Choose algorithm: FIFO, LRU, Optimal.
- Simulate and step through the replacement process.
- Light theme. No external libraries.

## How to use
1. Open `index.html` in a browser.
2. Enter reference string like `7,0,1,2,0,3,0,4`.
3. Set frames and choose algorithm.
4. Click Simulate. Use Step or Play to animate.

## Files
- `index.html` - main single-file app
- `style.css` - minimal styling
- `script.js` - simulation logic

Made for OSY micro-project. Minimal and self-contained.
